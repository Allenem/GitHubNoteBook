# 变量

## 使用

`变量格式: 名称 = 值`

```py
name = "老王"
print(name)  
#  输出：老王
```

## 变量命名规则

变量支持： **字母** 、 **数字** 、 **下划线** 组合。

注意：变量不能使用数字开头，变量区分大小写，以下划线开头的变量标识私有变量，不能被其他模块使用。

## 变量赋值

变量赋值支持多变量赋值，示例如下：

```py
name = "老王"
print(name)  
# 输出：老王
    
age, sex = 18, "男"
print(age)  
#  输出：18
    
x = y = 10
print(x)  
#  输出：10
```
